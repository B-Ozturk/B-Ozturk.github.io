Hallo welkom op mijn website. Ik ben Berke Öztürk en ben 16 jaar oud. Ik zit op het ROC Mondriaan en volg de opleiding Software developer. Vanwege de lock-down in maart 2020 kon ik niet meer naar school voor een tijdje en leek het me leuk om een website te maken en dat deed ik. De website dient als een platform waar je spelletjes kan vinden en ze kan spelen met je vrienden samen of tegen elkaar. Er zijn ook spelletjes te vinden die je kan spelen/downloaden voor op je telefoon zie hieronder de foto's van mijn website. Onder de foto's staat een knop waarmee je naar mijn website kan gaan. Ik zou het leuk vinden als je een kijkje neemt op mijn website en een review achter laat (een knop voor reviews komt eraan).

Mijn website

Op dit moment heb ik bij de Jumbo een bijbaantje als vakkenvuller. Later zou ik graag iets willen doen met het maken van website's omdat ik dan creatief bezig kan zijn. Ik zou eigenlijk een psycholoog willen worden maar toen ik op het mbo zocht naar een opleiding die daar een beetje te maken heeft vond ik het niet zo leuk want de opleiding trok me niet zo erg aan maar als ik de opleidingkan volgen op het hbo zou ik het graag doen. Ik had ook gekeken naar de opleiding elektrotechniek maar toen ik kennis maakte met de opleiding Software Developer en wist ik zeker dat ik voor deze opleiding zou kiezen omdat ik toen (nog steeds 😁) graag wilde leren programmeren.
